# 🟢 Presence Pulse

> A real-time web-based attendance and presence tracking system built with HTML, CSS, JavaScript, and Python.

---

## 📌 Overview

**Presence Pulse** is a lightweight, web-based presence and attendance tracking application. It provides an intuitive dashboard interface for monitoring real-time attendance, tracking presence status, and gaining actionable insights — all from a clean, responsive UI.

---

## ✨ Features

- 📊 **Live Attendance Dashboard** — View real-time presence and attendance data at a glance
- 👤 **Individual Tracking** — Monitor presence status for each person/employee
- 📅 **Daily & Monthly Trends** — Visualize attendance patterns over time
- 🐍 **Python Backend** — Lightweight server-side logic for data processing
- 🎨 **Responsive UI** — Clean, modern interface built with HTML & CSS
- ⚡ **Fast & Lightweight** — No heavy frameworks; runs efficiently in any browser

---

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed
- A modern web browser (Chrome, Firefox, Edge)

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/avishrakshe/presence-pulse.git
cd presence-pulse
```

2. **Navigate to the project folder**

```bash
cd "Presence pulse"
```

3. **Run the Python backend**

```bash
python app.py
```

4. **Open the application**

Open `index.html` in your browser, or visit the local server URL shown in the terminal.
